# MVP: PharmaPace PR - Core Analysis Engine

## Focus

The Minimum Viable Product concentrates on the **AI Fit Score** and the **True Cost of Living (TCOL)** projection.

## Key Deliverables

- **AI Fit Score**: A 0-10 metric evaluating how well a candidate's background (Validation, Engineering, Quality) aligns with the specific PR municipality's demand.
- **Commute/LUMA Engine**:
  - Calculate monthly delta in electricity based on destination municipality.
  - Project annual toll expenses using AutoExpreso standard rates for PR-22/PR-52.
  - Fixed Fuel Rate: $0.91/L (Jan 2026 baseline).
- **Recruiter Feedback**: 2-3 specific negotiation handles based on current market scarcity in PR.
- **W2 vs 1099 vs 480**: Essential tax-equivalent comparison for the PR market.
